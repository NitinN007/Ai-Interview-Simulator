WHAT IS THIS PROJECT?
================================================================================


How it works:
1. You tell the website what job you want (topic, job description, experience)
2. AI creates 5 interview questions for you
3. You record yourself answering on camera
4. AI watches your video and gives you a score (0-100)
5. AI gives feedback on how you did


WHO USES IT?
================================================================================
- Job interview candidates preparing for technical roles
- People practicing speaking skills
- Anyone wanting AI feedback on interview performance

KEY PROBLEMS SOLVED
================================================================================

Problem 1: AI code was breaking the website
Solution: Moved AI code to backend only (not client side)

Problem 2: AI response wasn't always JSON
Solution: Added backup way to read AI response

Problem 3: Couldn't send video over internet
Solution: Convert video to text format (base64) before sending

Problem 4: API quota limit (free tier)
Solution: Need to enable billing on Google Cloud (cheap - pennies per test)


HOW TO RUN IT
================================================================================

1. Download the code:
   git clone https://github.com/NitinN007/Ai-Interview-Simulator.git

2. Install dependencies:
   npm install

3. Get Google AI Key:
   Go to https://aistudio.google.com/app/apikeys
   Create new key
   Copy it

4. Create .env.local file with:
   GEMINI_API_KEY=paste_your_key_here
   NEXT_PUBLIC_GEMINI_API_KEY=paste_your_key_here
   NEXT_PUBLIC_DRIZZLE_DB_URL=your_database_url

5. Start website:
   npm run dev

6. Open in browser:
   http://localhost:3001


EXPLAINING TO INTERVIEWER
================================================================================

"I built an AI Interview Simulator. Here's what it does:

Users enter their interview topic and background. The app uses Google's AI
(Gemini) to generate 5 personalized interview questions. Then, users record
themselves answering those questions on camera. The AI watches the video,
analyzes their speaking clarity and confidence, gives a score from 0-100,
and provides feedback.

The backend uses Next.js server routes to safely call the AI (not from
browser). The frontend is built with React and Tailwind CSS for a clean,
responsive design.

The main challenges I solved were:
1. Moving AI calls to backend (server-only library)
2. Handling video as base64 text for HTTP transmission
3. Parsing AI responses with fallback logic

The app works end-to-end: questions ‚Üí recording ‚Üí analysis ‚Üí results."
